# Dự Án 3000
1. Sử dụng
  -  `` git clone https://github.com/hptprobook/3000.git``
2. Chi tiết
     * Admin:
       - `` cd admin``
       - `` npm i``
       - `` npm run dev``
     * Website:
       - `` cd website``
       - `` npm i``
       - `` npm run dev``
     * Server:
       - `` cd server``
       - `` php artisan serve``
       - `` php artisan migrate``
       - đổi tên .env.example thành .env
       - Cấu hình database( có thể chạy bằng xampp hoặc laragon) vào server/.env
