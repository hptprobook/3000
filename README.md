# Dự Án 3000
Dự án về website thương mại điện tử gồm 4 phần: server, admin, website, database
1. Sử dụng
   ``code`` git clone https://github.com/hptprobook/3000.git
2. Chi tiết
     * Admin:
       ``code`` cd admin
       ``code`` npm i
       ``code`` npm run dev
     * Website:
       ``code`` cd website
       ``code`` npm i
       ``code`` npm run dev
     * Server:
       ``code`` cd server
       ``code`` php artisan serve
       ``cod`` php artisan migrate
